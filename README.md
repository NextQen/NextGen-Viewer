# 🚀 NextGen Viewer - Premium YouTube Multi-View Dashboard

NextGen Viewer ek powerful aur premium multi-view tool hai jo users ko ek hi screen par ek sath multiple YouTube videos chalane ki suvidha deta hai. Yeh tool chote creators ke liye watch time analysis aur views management mein madad karne ke liye optimize kiya gaya hai.

---

## ✨ Features
- **⚡ Advanced Multi-Tab Support:** Ek click mein jitne chahein utne tabs open karein.
- **💎 Premium Glassmorphism UI:** Modern glow effect aur dark mode interface.
- **🔄 Auto-Loop Video Feature:** Videos automatically end hone par dobara shuru ho jayengi (Watch-time boost support).
- **🔇 Smart Autoplay:** Chrome aur modern browsers ki policies ke mutabik automatically mute hokar play hoga taaki playback block na ho.
- **🖥️ Fixed Desktop Layout:** Har device par professional desktop-grade workspace layout.

---

## 🛠️ Built With
- **HTML5** (Structure & SEO Meta Tags)
- **CSS3** (Custom Gradients & Glassmorphism design)
- **JavaScript** (Dynamic Iframe Injection & URL Extraction regex)

---

## 📖 How to Use
1. Apne YouTube video ka link copy karein.
2. Link ko **"Paste YouTube Link here..."** box mein dalein.
3. **"Tabs"** wale box mein number likhein ki aapko ek sath kitne videos chalane hain (Recommended: 1-20).
4. **"Play Videos"** button par click karein aur aapka dashboard ready ho jayega!

---

## 🔒 Performance & Optimization Note
Behtar performance ke liye ek baar mein 20-25 tabs se zyada na chalayein, taaki aapka browser smoothly run karta rahe.
